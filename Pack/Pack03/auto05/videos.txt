	2.Document format similar to HTML

Well formed XML

Displaying XML 
	adheres to basics structural requirements 
		1. Single root element 
		2. Matched tags, proper nesting 
		3. Unique attributes within elements 

-Use rule_based language to traslate to HTML	
	1. Cascading stylesheets (CSS)

-Valid XML
	Adheres to basic structural requirements

- Document Type Descriptor  ( DTD )
	1. Grammar-like language for specifying elements, attributes,nesting, ordering,#occurrences
	2. Also special attribute types ID and IDREF(S)
